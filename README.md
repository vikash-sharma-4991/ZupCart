🛒 ZupCart

ZupCart is a modern e-commerce web application built using React.js, Node.js, Express, and MongoDB.
It provides a seamless shopping experience where users can sign up, log in, browse products, filter by categories, add to cart, and checkout — all in a clean and responsive UI.

🚀 Features

🧑‍💻 User Authentication (Sign-up & Login with validation)

🛍️ Product Listing with images, prices, and offers

🔍 Search & Filter by product name or category

🛒 Add to Cart and manage cart items easily

💳 Checkout Page for order summary and confirmation

📱 Responsive Design — Works smoothly on all devices

⚡ Fast & Dynamic UI built with React

🌐 Backend Integration using Node.js & Express
